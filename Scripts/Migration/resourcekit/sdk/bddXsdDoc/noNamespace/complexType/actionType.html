Documentation

Defines a type of action that can be invoked from within a task.

Properties
This component is not nillable.

Attribute Detail

cancelTask

When this attribute is set to true, the task is canceled when this action is performed. Consequently, the task is completely deleted and not recoverable and any pending changes associated with the task will be permanently deleted.

Type:
xs:boolean
Default:
false
Use:
optional
Form:
unqualified

closeTaskView

When this attribute is set to true, the tab in which the BDD application user was working when performing this action will be closed, and the user will be returned to the Dashboard page.

manualReassign

When this attribute is set to true, the BDD application user is prompted to select a specific user for assignment of the task before the action is performed. This is used, for example, when reassigning a task to another user.
