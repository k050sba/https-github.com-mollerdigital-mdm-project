Attribute Detail

saName

The name of the match rule set subject area which will show control which rule set to use. Example: Person.

Type:
xs:string
Use:
optional
Form:
unqualified

type

The match type to use: One of the following values. AUTO - Use only auto match rules in the match rule set. BOTH - Use both auto and manual match rules in the match rule set. NONE - Use the search mode of the searchMatch API (not based on predefined match rules).
